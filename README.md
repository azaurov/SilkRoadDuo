# Silk Road Duo

An AI-powered language learning app for 7 ancient and classical languages of the Silk Road. Built with React Native + Expo SDK 56, runs on Android, and installable as a PWA on iPhone via Safari.

**Try it now:** [azaurov.github.io/SilkRoadDuo](https://azaurov.github.io/SilkRoadDuo/)

## Features

- **AI-generated lessons** — 20 exercises per lesson, freshly generated by Claude / Gemini / Groq each time
- **4 exercise types** — MCQ, fill-in-the-blank, matching pairs, word arrangement
- **Native script + romanization** — proper RTL rendering for Arabic/Hebrew/Farsi; Sogdian uses romanized transliteration (HarfBuzz limitation on Android)
- **TTS** — native speech for Arabic, Farsi, Hebrew, Uzbek (where device locale is installed)
- **Local user profiles** — multiple profiles per device, each with independent XP / streak / achievements, persisted via AsyncStorage across cold restarts
- **Gamification** — 3 hearts per lesson, 10 XP per correct answer, daily streak, 10 unlockable achievements
- **Background music** — Prince of Persia theme, pauses during lessons
- **PWA** — installable on iPhone via Safari → Share → Add to Home Screen; works offline after first load

## AI Backends

Set `EXPO_PUBLIC_LLM_BACKEND` to one of:

| Value | Provider | Notes |
|---|---|---|
| `openrouter` | OpenRouter | Default; routes to best available model |
| `groq` | Groq | Fast inference; free tier available |
| `gemini` | Google Gemini | Fallback in auto-chain |
| `local` | Ollama | `http://10.0.2.2:11434` (Android emulator host) |

Auto-fallback chain on 429: **Groq → OpenRouter → Gemini**

## Quick Start

```bash
git clone https://github.com/azaurov/SilkRoadDuo.git
cd SilkRoadDuo
npm install
cp .env.example .env   # add your API keys
npm start              # Expo dev server — scan QR with Expo Go
```

**.env keys:**
```
EXPO_PUBLIC_GROQ_API_KEY=...
EXPO_PUBLIC_GEMINI_API_KEY=...
EXPO_PUBLIC_OPENROUTER_API_KEY=...
EXPO_PUBLIC_LLM_BACKEND=openrouter
```

**EAS cloud builds (recommended):**


## Architecture

All app logic lives in a single file: **`App.js`**. No routers, no state-management libraries.

**Screen flow:** `home → topic → loading → lesson → result` (plus `home → achievements`)

**State owner:** root `App` component holds `screen`, `activeLang`, `activeTopic`, `exercises`, `stats`, `resultData`, and all profile state.

**Profile storage:** `assets/lib/profiles.js` — AsyncStorage at `@silk-road-duo:profiles` / `@silk-road-duo:activeId`. Atomic writes via a serialized write queue prevent race conditions when stats and achievements are saved simultaneously after lesson completion.

**Web PWA:** `expo export --platform web` outputs to `dist/`. `scripts/patch-web-pwa.js` injects manifest link, apple meta tags, and service worker registration into the generated `index.html`. Deployed to GitHub Pages via `.github/workflows/deploy-web.yml` on every push to master.

```
  lib/
```
